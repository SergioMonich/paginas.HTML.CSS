# paginas.HTML.CSS
Treinamento Prático HTML & CSS
Projeto acadêmico para prática de tags HTML específicas e propriedades CSS obrigatórias.


📋 Sobre o Projeto
Este repositório contém 15 exercícios que treinam conceitos fundamentais de HTML e CSS. Cada exercício utiliza uma tag HTML obrigatória e uma propriedade CSS obrigatória, conforme especificado nos requisitos.
